# Xbench 误报清理工具 — 操作指南

本工具用于自动删除 Xbench QA 报告中已确认的误报条目，无需手动逐行删除。

---

## 准备工作（首次使用）

### 第一步：安装 Python

1. 打开浏览器，访问 https://www.python.org/downloads/
2. 点击黄色的 **Download Python** 按钮下载安装包
3. 运行安装包，**必须勾选底部的 "Add Python to PATH"**，然后点击 Install Now
4. 安装完成后关闭窗口

> 以后使用不需要重复此步骤。

---

## 每次使用的操作步骤

### 第一步：准备两个文件

| 文件 | 说明 |
|------|------|
| **对照文件** | 审校人员已在 Comments 列填写了"误报"、"FP"等标注的 Xbench 报告 |
| **待处理文件** | 需要清理误报条目的 Xbench 报告（通常是最新版本） |

两个文件可以放在任意位置，不需要和脚本在同一个文件夹。

---

### 第二步：双击运行 `run.bat`

找到本工具文件夹中的 **`run.bat`** 文件，双击打开。

出现黑色窗口，第一次运行会自动安装所需的程序组件（需要联网，约需 1 分钟）。

---

### 第三步：输入文件路径

窗口会依次提示输入两个文件的路径。

**最简单的方法是拖拽文件**：
1. 窗口显示"对照文件"的提示后，将**对照文件**直接拖入黑色窗口
2. 路径自动填入后，按回车
3. 窗口显示"待处理文件"提示后，将**待处理文件**拖入，按回车

---

### 第四步：等待完成

工具自动处理，完成后显示：

```
================================================
  [完成] 已生成清理后文件（文件名含 _cleaned）
================================================
```

---

### 第五步：获取结果

清理后的文件保存在**待处理文件的同一文件夹**中，文件名末尾会加上 `_cleaned`。

例如：
- 输入：`Xbench_QA_异世界fr V2.xls`
- 输出：`Xbench_QA_异世界fr V2_cleaned.xls`

原文件**不会被修改**。

---

## 常见问题

**Q：窗口一闪而过，什么都没看到？**
右键点击 `run.bat`，选择"以管理员身份运行"。

**Q：出现"未找到 Python"错误？**
安装 Python 时没有勾选 "Add Python to PATH"，需要重新安装并勾选该选项。

**Q：出现"No Excel COM available"错误？**
`.xls` 格式需要电脑上安装 Microsoft Excel 或 WPS Office。确认已安装后重试。

**Q：处理结果显示"0 组误报"？**
检查对照文件的 Comments 列，确认已填写"误报"或"FP"等关键词。

**Q：需要保存结果到指定位置？**
联系技术人员，可通过命令行参数 `--output` 指定输出路径。

## 误报标记规范

在 Xbench 报告的 **Comments 列**填写以下任意内容即可被识别：

| 标注内容 | 是否识别 |
|----------|----------|
| 误报 | ✅ |
| FP | ✅ |
| fp | ✅（不区分大小写） |
| False Positive | ✅ |
| FE | ✅ |
| 误报，已确认 | ✅（含关键词即可） |
| 正常 | ❌ |
| OK | ❌ |
